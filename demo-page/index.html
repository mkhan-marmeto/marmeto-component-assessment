Welcome to the Design System Demo page. Here you will find examples of reusable components, their descriptions, and guidelines on how to apply specific classes to achieve desired styles.

Button Component

Use the following classes to achieve various button styles:

  • .btn.btn-primary: Primary button styling.
  • .btn.btn-secondary: Secondary button styling.
  • .btn.btn-disabled: Disabled button styling (requires disabled attribute).

Card Component

Use the following structure and classes to create card components:

  • .card: Main container for the card.
  • .card-img: Optional image inside the card.
  • .card-title: Title text of the card.
  • .card-subtitle: Subtitle text of the card.
  • .card-body: Main body text of the card.

Input Component

Use the following structure and classes to create input fields:

  • .input-field: Wrapper for the input field and its related elements.
  • .input-default: Default input field styling.
  • .input-error: Error message styling (visible only when there's an error).
